# Student Portfolio Website

## Overview
This is a static personal portfolio website developed to showcase a student’s academic background, technical skills, projects, achievements, and contact information for placement purposes.

The website is built using only HTML and CSS and demonstrates structured page design, consistent navigation, and clean presentation.

---

## Features
- Personal profile presentation
- Academic qualifications section
- Technical skills overview
- Project showcase
- Achievements and certifications
- Contact information page
- Consistent layout across all pages
- Profile photo displayed on pages (except homepage)

---

## Technologies Used
- HTML5
- CSS3
